Abril Insurance Products

How we used a new product to start a holistic process of design in a growing team.

Brief

This product was a partnership between a book publisher Abril and Mongeral Aegon that looked for selling insurance products based on each magazine target audience.

Abril project was used as an opportunity to promote the structuring of a design process inside the team.

It was essencial for us to start testing and validating ideas as soon as possible in a team that was currently in a stage of full delivery focus.

We applied a framework built and adapted for us called PRISMA Framework.

Sketching

Sketching the product through a sketch rounds with voting for the best ideas.

The solution

Progress of solution through three steps: sketch, wireframe and user interface.

Testing

Usability tests in each framework cycle.

Testing as early as possible / Simple flow testing / Validating the idea.

Testing the improvements acquired from sketch test as well the purchase flow testing.

High-fidelity navigation testing, including entire flow, value proposal and UI components.
